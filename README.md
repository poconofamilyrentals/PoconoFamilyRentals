# Pocono Family Rentals Website

Static website for Pocono Family Rentals, ready to publish with GitHub Pages.

## Files

- `index.html` - main website
- `gallery.html` - full photo gallery
- `styles.css` - shared styling
- `script.js` - small header scroll behavior
- `.nojekyll` - tells GitHub Pages to serve the site as plain static files

## Publish With GitHub Pages

1. Create a GitHub repository.
2. Upload the website files to the repository root:
   `index.html`, `gallery.html`, `styles.css`, `script.js`, `.nojekyll`, and `README.md`.
3. In GitHub, go to `Settings` -> `Pages`.
4. Under `Build and deployment`, set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Save. GitHub will publish the site at:
   `https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/`

## Updating

Edit the files locally, then upload or commit the changed files to GitHub. GitHub Pages will redeploy automatically after each update to the selected branch.
